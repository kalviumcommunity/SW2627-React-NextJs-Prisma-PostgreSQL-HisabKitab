Project name - Hisab kitab
Problem it is solving - Khatabook wants a digital ledger where shopkeepers add transactions and see a running balance that updates immediately. The transaction history is paginated, and each transaction can be edited/deleted with an audit trail. Multiple shopkeepers cannot edit the same transaction simultaneously.
